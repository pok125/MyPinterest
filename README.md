# MyPinterest
## 프로젝트 개요
* Django를 이용하여 Pinterest서비스를 참고하여 만들어 배포까지 목표
## 프로젝트 목표
* generic view를 이용하여 빠르게 전체적인 기능 구현 (SSR)
* 일반 view로 전체코드 리팩토링 후 배포 (SSR)
* DRF를 이용하여 REST API 구현
